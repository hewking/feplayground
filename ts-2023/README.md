# ts-2023

2023 年如何配置typescript

## 步骤

1. 配置package.json
   1. npm init -y
   2. set type to "module"
2. 安装typescript
   1. npm install -D typescript
   2. npm i -D @types/node

## tsconfig 配置

1. module 配置 `NodeNext`
